# Vibe Coding Prompts

### Step 1: The Shell
"Build a single-page mobile web app in index.html and style.css. The design must be a pixel-perfect recreation of the Winamp 2.0 skin. Use a dark theme with neon green accents. Ensure it is responsive for a mobile vertical view."

### Step 2: The Player Logic
"Add script.js to handle audio playback. Create a hardcoded JSON playlist of 5 songs. Implement Play, Pause, Next, and Previous functionality. Ensure the 'current track' is highlighted in the playlist UI."

### Step 3: The 'Vibe' Polish
"Add a CSS animation to the visualizer area so it 'dances' when music is playing. Make the volume slider look like the silver Winamp slider. Use beveled borders on all container elements to give it that 90s software feel."
